# MiniApp Usuários

Interface para bootstrap do administrador e gestão de contas. Este MiniApp depende do módulo `miniapp-base/js/auth.js` e do adaptador `miniapp-base/js/adapters/users-appscript.js` para comunicar com o backend em Apps Script. O arquivo `index.html` possui duas visões: configuração inicial do administrador (`mode=setup`) e painel de gestão (requer papel `admin`).
