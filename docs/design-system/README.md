# Design System – MiniApp 5Horas

Esta pasta será usada para centralizar o **Design System** da plataforma MiniApp 5Horas.

Aqui ficarão:

- documentação de componentes visuais (botões, tabelas, cards, indicadores);
- CSS compartilhado para estilos globais de UI;
- qualquer guia de uso dos componentes em MiniApps.

Nesta fase, o Codex **NÃO DEVE** criar novos componentes nem novas regras de CSS aqui.
Apenas esta estrutura inicial e este README.

Fases futuras irão adicionar:

- arquivos CSS (ex.: `tokens.css`, `buttons.css`, `tables.css`);
- documentação de componentes com exemplos.
