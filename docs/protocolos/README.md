# Protocolos operacionais

Este diretório centraliza procedimentos que devem ser seguidos antes de alterações sensíveis no ecossistema de MiniApps.

## Como usar
1. Leia os protocolos disponíveis antes de iniciar qualquer tarefa para confirmar se há instruções aprovadas para o fluxo desejado.
2. Caso um protocolo não exista, documente o passo a passo após alinhamento com a equipe responsável e registre o novo arquivo aqui.
3. Atualize o `CHANGELOG.md` sempre que um protocolo for criado ou revisado.
