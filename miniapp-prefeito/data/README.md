# miniapp-prefeito/data/

Fontes de dados locais utilizadas pelo MiniApp Prefeito.

## Arquivos
- `sample.json` – conjunto de dados de exemplo usado como fallback offline e para demonstrações rápidas.

Caso sejam adicionados novos arquivos (CSV, JSON, etc.), documente aqui a finalidade e lembre-se de atualizar o Service Worker (`sw.js`) se precisarem estar disponíveis offline.
