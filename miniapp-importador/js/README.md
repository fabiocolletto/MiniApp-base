# Scripts do Importador de Pesquisas

Contém a lógica do fluxo de importação, incluindo interação com IndexedDB, estado simulado do backend e tradução das mensagens.

- `importador.js`: script principal executado pela página `../index.html`.

Sempre que novas funcionalidades forem adicionadas, documente-as aqui e garanta cobertura i18n para `pt-BR`, `en-US` e `es-ES`.
