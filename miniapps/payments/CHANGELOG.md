# Changelog - MiniApp Pagamentos e Assinaturas

## [0.2.0] - 2025-02-16
### Adicionado
- Snapshot simplificado reutilizado pelo MiniApp Configurações para exibir status de assinaturas e atalhos antes de abrir este painel completo.
- Documentação atualizada descrevendo o vínculo entre Configurações e o painel dedicado.

### Adicionado
- Estrutura inicial do MiniApp com card compacto/expandido dedicado ao Brasil via Mercado Pago.
- Sessões "Plano e cobrança", "Histórico resumido" e "Ações" com dados mockados e botões que apontam para o Mercado Pago.
- Script `payment-control.js` com hidratação de dados e comportamento responsivo de expansão do card.
