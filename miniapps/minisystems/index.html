MiniSystems

Preferências globais do usuário

O MiniSystems controla as preferências que são aplicadas imediatamente em toda a plataforma. O cartão embutido no painel de Configurações já salva o tema claro/escuro e o idioma favorito e envia estes dados para o shell via postMessage assim que o usuário seleciona uma opção.

  • Preferências salvas em localStorage com fallback para valores padrão.
  • Eventos miniapp:global-preferences enviados localmente e para o shell.
  • Integração direta com o monitor de dados do usuário e o painel de pagamentos.
