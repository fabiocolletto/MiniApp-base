# Changelog - MiniApp Gestão de Catálogo

## [0.2.0] - 2025-02-17
### Adicionado
- Grid responsivo em React + Material UI dentro do `index.html`, utilizando `AppCard/AppButton/AppSection` e o novo `AppModalProvider`.
- Dialog de detalhes para cada MiniApp (categoria, contrato, responsável) acionado a partir de cada card.

### Alterado
- Layout anterior em tabela HTML substituído por cards de 320px e `docs/miniapp-global.css` ajustado para servir apenas como base de tokens/layout.
