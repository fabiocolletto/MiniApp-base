# Pasta `templates/miniapps-archive/`

Esta pasta é opcional e pode ser usada para armazenar templates de MiniApps já processados.

O Codex só deve mover arquivos para esta pasta **após**:

- criar o MiniApp definitivo dentro da estrutura oficial (`apps/<slug>/`), e
- registrar a entrada em `docs/miniapp-data.js`.

Em caso de dúvida, é preferível manter o template em `miniapps-inbox/` até confirmação manual.
