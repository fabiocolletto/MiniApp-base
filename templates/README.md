# Pasta `templates/`

Esta pasta armazena arquivos temporários usados como modelos para criação de novos MiniApps.

Os templates **não fazem parte da execução da PWA em produção**. Eles servem apenas para orientar o Codex na geração de novos MiniApps dentro da estrutura oficial do projeto.

Subpastas:

- `miniapps-inbox/`: templates aguardando processamento.
- `miniapps-archive/`: (opcional) templates já processados, caso sejam mantidos para histórico.
