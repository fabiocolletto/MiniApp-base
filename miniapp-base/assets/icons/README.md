# Ícones

Este diretório reúne os ícones utilizados pelos cartões do catálogo de miniapps. Cada arquivo segue o formato 56×56 px pensado para se ajustar ao layout responsivo dos cards.

- `prefeito.png`: ícone do Painel do Prefeito, com o brasão estilizado utilizado na versão inicial do miniapp.
- `survey.svg`: ícone vetorial do Importador de Pesquisas, representando uma prancheta com seta de importação.
