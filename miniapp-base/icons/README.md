# miniapp-base/icons/

Este diretório abriga os ícones utilizados pelo manifesto PWA do shell.

## Como proceder
- Adicione aqui, manualmente, as versões finais dos ícones `icon-192.png` e `icon-512.png` assim que estiverem disponíveis.
- Ambos devem ser arquivos PNG quadrados nos tamanhos 192×192 e 512×512 pixels, respectivamente.
- Após incluir os arquivos, mantenha os mesmos nomes e limpe o cache do manifesto no navegador ou atualize o número da versão do Service Worker se necessário.

> **Nota**: Os ícones não são distribuídos junto com o pacote porque o fluxo atual não suporta envio de binários pelo agente. Utilize este lembrete como texto de espera até que os assets oficiais sejam adicionados manualmente.
