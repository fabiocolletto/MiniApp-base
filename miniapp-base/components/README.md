# Componentes do MiniApp Base

Este diretório continua reservado para componentes visuais e experimentos específicos do design system. Os utilitários JavaScript compartilhados foram movidos para `../js/`.

Ao alterar ou adicionar componentes, mantenha a compatibilidade com os tokens definidos em `miniapp-base/style/styles.css`, atualize o `CHANGELOG.md` e garanta traduções completas para os idiomas suportados.
