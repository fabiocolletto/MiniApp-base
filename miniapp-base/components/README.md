# Componentes do MiniApp Base

Este diretório concentra os protótipos e utilidades específicos utilizados pelo MiniApp Base.

- `carousel.js`: script compartilhado para os carrosséis utilizados em catálogos e dashboards.

O fluxo completo de importação de pesquisas agora vive no miniapp dedicado `../miniapp-importador`.

Ao alterar ou adicionar componentes, mantenha a compatibilidade com os tokens definidos em `miniapp-base/style/styles.css`, atualize o `CHANGELOG.md` e garanta traduções completas para os idiomas suportados.
