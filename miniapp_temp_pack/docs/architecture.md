# Estrutura do Repositório

```
/
├─ index.html
├─ README.md
├─ AGENTE.md
├─ CHANGELOG.md
├─ public/
│  ├─ sw.js
│  └─ manifest.json
├─ src/
│  ├─ core/
│  ├─ modules/
│  ├─ miniapps/
│  ├─ ui/
│  └─ app/
└─ docs/
```

A raiz contém a entrada do GitHub Pages.  
`TEMP_INBOX/` **não deve existir em produção** — é só pasta temporária de ingestão.
