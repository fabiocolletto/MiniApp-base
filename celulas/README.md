Esta pasta abriga apenas as células compatíveis com o Genoma v1.2.

- **home/**: célula fixa registrada pelo Genoma.

Regras ativas:
- Cada célula mantém `index.html`, registra-se via `window.PWAO_RegistrarCelula` e não acessa arquivos externos ao seu diretório.
- Não há células legadas ou dependências de frameworks SPA.
- Novas células só entram após seguirem o Template Oficial de Célula Fixa v1.
